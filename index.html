Healthcare Service Quality Analysis

Statistical Analysis using Python

View Python Notebook

Project Overview

This project analyzes factors affecting hospital service quality using statistical methods. The goal is to identify key variables influencing quality scores and provide actionable insights.

Objectives

  • Evaluate factors affecting hospital service quality
  • Compare quality scores across different staff groups
  • Identify statistically significant differences
  • Provide insights to improve service performance

Methodology

  • Data cleaning and preprocessing using Python
  • Exploratory data analysis (EDA)
  • Statistical testing using Kruskal-Wallis test
  • Evaluation based on p-values (significance level 0.05)

Key Insights

  • Working hours significantly impact quality scores, showing different patterns across groups.
  • Staff in different departments exhibit varying perceptions of service quality.
  • Direct interaction with patients strongly influences satisfaction scores.
  • Job roles affect evaluation outcomes, reflecting differences in workload and responsibilities.
  • Statistical testing confirms multiple factors have significant effects (p-value < 0.05).

Business Impact

The analysis highlights that workload distribution, staff roles, and patient interaction are critical drivers of service quality. Optimizing staffing strategies and reducing excessive working hours can improve both employee experience and patient satisfaction.

Tools Used

  • Python (Pandas, NumPy)
  • Scipy (Statistical Testing)
  • Jupyter Notebook
