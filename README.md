# Healthcare Service Quality Analysis (Python)
 Statistical analysis project focusing on healthcare service quality evaluation.
 
This project analyzes factors affecting hospital service quality using statistical methods in Python. The goal is to identify key drivers influencing quality scores and provide actionable insights.

## Overview
- Dataset: Hospital staff survey data
- Focus: Service quality evaluation (a1 → a18)
- Method: Statistical testing (Kruskal-Wallis)
- Tool: Python (Pandas, Scipy)

## Key Insights
- Working hours significantly impact quality scores, showing variation across different workload groups  
- Staff roles and departments influence service quality perception  
- Direct patient interaction is a key driver of satisfaction scores  
- Multiple factors show statistically significant differences (p-value < 0.05), confirming variability in quality perception  

## Business Impact
The analysis highlights that workload, staff roles, and patient interaction strongly influence service quality.  
Optimizing staffing allocation and reducing excessive working hours can improve both employee experience and patient satisfaction.
