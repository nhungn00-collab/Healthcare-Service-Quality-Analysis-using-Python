# Healthcare Service Quality Analysis (Python)

This project analyzes factors affecting hospital service quality using statistical methods in Python.

## Overview
- Goal: Identify key factors influencing quality scores
- Method: Statistical testing (Kruskal-Wallis)
- Tool: Python (Pandas, Scipy)

## Key Insights
- Working hours significantly impact quality scores
- Staff roles and departments influence service perception
- Direct patient interaction strongly affects satisfaction
- Several factors show statistically significant differences (p-value < 0.05)

## Business Impact
Optimizing workload distribution and improving frontline support can enhance service quality and patient satisfaction.

## Tools Used
- Python (Pandas, NumPy)
- Scipy (Statistical Testing)
- Jupyter Notebook
